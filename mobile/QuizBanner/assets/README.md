# Asset files needed for the React Native app

The following asset files are needed in the /assets folder:

1. **icon.png** (1024x1024) - App icon
2. **adaptive-icon.png** (1024x1024) - Android adaptive icon
3. **splash.png** (1284x2778) - Splash screen image
4. **favicon.png** (32x32) - Web favicon

You can create simple placeholder icons or use a tool like:
- https://www.appicon.co/
- https://icon.kitchen/
- Figma/Canva for custom designs

For now, the app will work without these assets, but you'll see warnings.
